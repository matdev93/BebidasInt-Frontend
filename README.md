# Descripcion del proyecto:

El proyecto 5 consiste en una aplicacion de comercio electrónico de una tienda de bebidas. Se creo usando las siguientes tecnologias:

- HTML5
- CSS3
- Bootstrap
- MUI
- JavaScript
- ReactJS y React Router
- ExpressJS
- MongoDB
- Mongoose
